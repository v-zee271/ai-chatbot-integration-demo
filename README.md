# AI Chatbot Integration & Technical Documentation Demo

This project serves as a professional portfolio demonstration for AI automation and technical writing services. It is designed to showcase the quality of work provided through Fiverr gig offerings.

## 🚀 Overview

The demo features a landing page that highlights:
- **Strategic Service Tiers**: Clear pricing and feature breakdown for AI integration projects.
- **RAG (Retrieval-Augmented Generation) Proof**: Live examples of documentation (API Reference & User Guides) used to train custom AI agents.
- **Modern UI/UX**: A responsive, high-performance interface built with Tailwind CSS and glassmorphism design principles.

## 🛠 Tech Stack

- **Frontend**: HTML5, Tailwind CSS.
- **Typography**: Inter via Google Fonts.
- **Hosting**: Optimized for **DigitalOcean** (Static Site).

## 📁 Project Structure

- `index.html`: The main landing page and portfolio hub.
- `main.css`: Centralized styles for documentation components and UI elements.
- `api2.html` / `quickstart.html`: Dedicated documentation pages for training samples.
